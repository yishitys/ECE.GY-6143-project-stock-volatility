## 数据格式

每个CSV文件包含以下列：
- `timestamp`: 时间戳（UTC时区）
- `open`: 开盘价
- `high`: 最高价
- `low`: 最低价
- `close`: 收盘价
- `volume`: 成交量

## 数据说明

1. **时间粒度**: 每小时数据（从每日数据扩展而来）
2. **时区**: 所有时间戳已转换为UTC，与Reddit数据对齐
3. **交易时间**: 美国股市交易时间（9:30 AM - 4:00 PM EST）内的每小时数据点
4. **数据范围**: 2021年全年（2021-01-01 至 2021-12-31），与Reddit数据时间范围一致

## 使用方法

使用 `src/data_loading/load_stock_data.py` 脚本下载数据：

```bash
# 下载所有高优先级股票
python src/data_loading/load_stock_data.py --priority high

# 下载所有股票
python src/data_loading/load_stock_data.py --priority all

# 下载指定股票
python src/data_loading/load_stock_data.py --symbols GME AMC TSLA

# 自定义日期范围
python src/data_loading/load_stock_data.py --symbols GME --start-date 2021-01-01 --end-date 2021-12-31
```

## 注意事项

1. HOOD的数据从2021-07-29开始，因为它是2021年IPO的股票
2. 数据从每日数据扩展为每小时数据，每个交易日包含8个数据点（9:30 AM, 10:00 AM, ..., 4:00 PM EST）
3. 非交易日（周末、节假日）没有数据
